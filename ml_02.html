1. train_test_split()설명

기본문법 : train_test_split(arrays, test_size, train_size, random_state, shuffle, stratify)
arrays : 데이터와 레이블이 포함된 numpy 배열이나 pandas DataFrame입니다.
test_size : 테스트 데이터셋의 크기를 결정합니다.
이 값은 0과 1 사이의 실수로 주어집니다.
예를 들어, test_size=0.2로 지정하면 전체 데이터셋의 20%를 테스트 데이터셋으로 사용합니다.
train_size : 학습 데이터셋의 크기를 결정합니다.
test_size와 마찬가지로 0과 1 사이의 실수로 주어집니다.
기본값은 None으로, 학습 데이터셋 크기를 1 - test_size로 결정합니다.
random_state : 데이터를 나눌 때 사용되는 난수 시드입니다.
이 값을 지정하지 않으면, 매번 실행할 때마다 다른 결과를 얻을 수 있습니다.
shuffle : 데이터를 섞을지 여부를 결정합니다. 기본값은 True입니다.
stratify : 레이블을 고려하여 데이터를 나눌지 여부를 결정합니다.
예를 들어, stratify=y로 지정하면 레이블 데이터 y에 따라 학습 데이터셋과 테스트 데이터셋의 클래스 비율이 유지됩니다.
Return: :
X_train, X_test, Y_train, Y_test : arrays에 데이터와 레이블을 둘 다 넣었을 경우의 반환이며, 데이터와 레이블의 순서쌍은 유지된다.
X_train, X_test : arrays에 레이블 없이 데이터만 넣었을 경우의 반환

3. 머신 러닝의 학습 방법

지도 학습 : 입력 데이터와 이에 상응하는 출력 데이터(결과 또는 레이블)를 사용하여 모델을 훈련하는 방법.(입력 데이터와 결과 데이터를 컴퓨터에 입력하고, 이를 컴퓨터가 학습하여 유사한 데이터가 입력되었을 경우 결과를 예측하게 하는 방법) 예를 들어, 이메일이 스팸인지 아닌지를 분류하는 스팸 필터가 지도 학습에 해당합니다.

비지도 학습 : 출력 데이터(결과 또는 레이블) 없이 입력 데이터의 패턴이나 군집을 찾아내어 결과를 예측하는 방법. 예로 고객의 구매 패턴을 분석해 그룹화하는 작업이 있습니다.

강화 학습 : 에이전트(머신러닝 알고리즘)에 상벌을 추가하여 학습하게 하는 방법

4. 머신 러닝 알고리즘

선형 회귀 : 독립(입력) 변수와 종속(대상) 변수간의 상관 관계를 모델링하는 기법. 예를 들면 독립 변수라는 아파트의 평수 , 위치, 방 수 및 기타 특징에 따라 종속 변수인 집값이 예측/결정된다.

로지스틱 회귀 : 결과를 예측하는 것이 목적인 이항 분류 문제에 사용됩니다. 로지스틱 회귀에서는 입력 변수의 선형 모델을 기반으로 대상 변수의 확률을 추정합니다. 신청자의 신용 점수 및 기타 재무 데이터를 기반으로 대출 신청이 승인될 것인지 여부를 예측하는 것이 한 예입니다.
